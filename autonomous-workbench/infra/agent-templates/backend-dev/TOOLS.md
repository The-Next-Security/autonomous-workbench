# TOOLS.md — Backend Developer

## Skills
- `backend-developer` (propia).
- `coding-agent` para trabajo largo de código.
- `governance-wrapper`, `github-manager`.

## Stack operativo
- Node.js 22+ con TypeScript estricto.
- Frameworks: Fastify (preferido), Express, Hono.
- ORM: Prisma o Drizzle.
- Tests: vitest + supertest + testcontainers.

## Comandos frecuentes
- `pnpm dev` / `npm run dev`.
- `pnpm test:int` para integración.
- `pnpm db:migrate`.
- `docker-compose up -d` para stack local.
