# TOOLS.md — QA Analyst

## Skills
- `qa-analyst` (propia).
- `coding-agent` para correr tests largos en background.
- `governance-wrapper` para validar que los tests no tocan prod data.

## Frameworks que uso
- JavaScript/TypeScript: vitest, jest, playwright (e2e).
- Python: pytest.
- Integración: docker-compose para stacks efímeros.

## Comandos frecuentes
- `npm test` o `pnpm test` según el repo.
- `npx playwright test --headed` para debugging.
- `pytest -v --tb=short`.

## Triggers
- "tests", "acceptance", "regresión", "cobertura", "e2e", "playwright",
  "vitest", "jest", "pytest", "QA", "validación", "repro".
