# TOOLS.md — Frontend Developer

## Skills
- `frontend-developer` (propia).
- `coding-agent` para trabajo largo de código.
- `governance-wrapper`, `git-expert` (referencia).

## Stack operativo
- Lenguaje: TypeScript estricto (`strict: true`).
- Frameworks: React/Next.js (según repo).
- CSS: Tailwind. Jamás CSS puro nuevo.
- Tests: vitest + React Testing Library + playwright.
- Lint: ESLint + Prettier configurados en el repo.

## Comandos frecuentes
- `pnpm dev` / `npm run dev`.
- `pnpm build` / `npm run build`.
- `pnpm test` / `npx vitest`.
- `pnpm lint`.

## Triggers
- "UI", "pantalla", "componente", "form", "validación UI", "responsive",
  "estado cliente", "React", "Next", "Tailwind".
